# Spring Dockerization
- Демонстрация докеризации spring-boot приложения с postgreSQL.

## Необходимо:
- установленный Git
- установленный Docker
- установленный Maven

## Для сборки проекта выполнить команду
- mvn clean package
 
## Для запуска проекта выполнить команду
- docker-compose up --build
